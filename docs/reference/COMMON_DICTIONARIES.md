**DL-автор:** перед созданием нового словаря с тем же смыслом откройте **нужный раздел** `## \`имя\`` — не читайте весь файл.

Правила оформления: `docs/D_GENERATION_RULES.md`, `docs/A_LANGUAGE_SPEC.md` §3.
В правилах узла: `[dict(common_yes)]`, в ответах с нормализацией: `[&dict(last_names, norm)]`.
