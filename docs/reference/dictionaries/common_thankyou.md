#common_thankyou

**Core intent:** Выражение благодарности: «спасибо», «благодарю», «thanks», «сенкью» и все опечатки/варианты написания.

**Intent types:** affirmation, gratitude

**Usage:** standalone `[common_thankyou]` / в `[@cmb]` как триггер благодарности

**Edge cases / warnings:**
- Содержит формы с тильдой `~` (опечатки-варьировки)
- Смешанные регистры и раскладки (например `cpasibo` — русская «спасибо» на английской раскладке)
- Содержит фразы на других языках: `thanks`, `thank`, `аригато`, `данке`, `дякую`, `мерси`, `рахмэт`
- Пустые строки (key `85s0g`, `hao6`, `do91b`, `7lll8`) — артефакты экспорта, можно игнорировать
