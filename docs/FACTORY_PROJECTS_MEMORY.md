# Память о проектах фабрики

Файл фиксирует **состояние каждого бота в работе**, чтобы генерацию можно было прервать и в любой момент возобновить с того же места.

> **Общие правила фабрики** — `docs/PROJECT_MEMORY.md`.  
> **Рабочие файлы** — `work/<BotSlug>/`

При возобновлении **сначала читайте** этот файл и поле **«Следующий шаг»**.
