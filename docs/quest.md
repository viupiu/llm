**Мир квеста: Долина Туманных Руин**

**Назначение**

Этот документ задает основу мира для проектирования большого текстового квеста.

Цель проекта — создать длинное прохождение (примерно 15–20 игровых шагов) без взрывного роста количества веток.

Основной принцип:

- ветки периодически сходятся;
- решения игрока влияют на доступные возможности;
- существуют разные концовки;
- мир выглядит большим, даже если граф остается компактным.

**Завязка**

Игрок приходит в Долину Туманных Руин.

По легенде, где-то в центре долины находится Древнее Сердце — артефакт неизвестного происхождения.

Одни считают его источником силы.

Другие — причиной всех бедствий региона.

Игрок может:

- попытаться уничтожить Сердце;
- забрать его себе;
- покинуть долину;
- погибнуть по дороге.

**Основные локации**

**1\. Опушка леса**

Стартовая зона.

Игрок получает первое решение:

- идти через лес;
- идти по старой дороге.

Обе ветки позже сходятся.

**2\. Лес Шепчущих Корней**

Особенности:

- странные голоса;
- потерянные путники;
- лесные духи.

Возможные события:

- найти травника;
- заблудиться;
- получить подсказку о Сердце.

**3\. Старая Дорога**

Особенности:

- разрушенные телеги;
- следы разбойников;
- заброшенная часовня.

Возможные события:

- встретить торговца;
- найти карту;
- подвергнуться нападению.

**4\. Перекресток Воронов**

Первая точка слияния сюжетов.

Сюда можно попасть:

- через лес;
- через дорогу.

Игрок понимает, что путь к центру долины проходит через горы.

**5\. Заброшенная Деревня**

Особенности:

- пустые дома;
- колодец;
- староста-призрак.

Можно:

- искать припасы;
- разговаривать с призраком;
- исследовать дома.

Некоторые решения дают знания или предметы.

**6\. Болото Светляков**

Опасная зона.

Возможности:

- короткий путь;
- безопасный обход.

Короткий путь может привести к неприятностям.

**7\. Каменный Мост**

Вторая точка слияния.

Практически все маршруты приводят сюда.

Мост охраняет существо.

Не обязательно сражаться.

Можно:

- договориться;
- обмануть;
- пройти силой.

**8\. Горный Проход**

Здесь игрок выбирает:

- пещеры;
- перевал.

Обе ветки ведут к крепости.

**9\. Хрустальные Пещеры**

Особенности:

- древние механизмы;
- загадки;
- подземное озеро.

Можно найти важную информацию о Сердце.

**10\. Перевал Ветров**

Особенности:

- опасные обрывы;
- наблюдательная башня;
- следы экспедиции.

Более безопасный маршрут.

Меньше наград.

**11\. Крепость Без Короля**

Крупная точка слияния.

Сюда приходят все основные маршруты.

Игрок узнает происхождение Сердца.

После этого начинается финальная часть сюжета.

**12\. Подземный Храм**

Последняя большая локация.

Содержит:

- испытания;
- хранителей;
- древние записи.

**13\. Зал Сердца**

Финальная зона.

Игрок сталкивается с последствиями всех предыдущих решений.

**Возможные существа**

Использовать умеренно.

Не превращать квест в непрерывную боевку.

**Лесные духи**

Не злые.

Любят загадки.

**Болотный Страж**

Охраняет болото.

Может пропустить игрока.

**Каменный Страж Моста**

Проверяет достойных путников.

**Пещерные Тени**

Создают ложные пути.

**Хранители Храма**

Последнее препятствие.

**Типы решений**

Не все решения должны быть бинарными.

Допустимые варианты:

**Большой выбор**

2–3 варианта.

Использовать на крупных развилках.

**Малый выбор**

Да / Нет.

Использовать часто.

**Продолжение сюжета**

Один вариант:

- идти дальше;
- открыть дверь;
- спуститься вниз.

Допустимо для сохранения темпа.

**Рекомендуемая структура**

Шаги 1–5:  
Исследование мира.

Шаги 6–10:  
Опасности и выбор маршрутов.

Шаги 11–15:  
Раскрытие тайны Сердца.

Шаги 16–20:  
Финал и последствия решений.

**Концовки**

Минимум 4.

**Хорошая**

Сердце уничтожено.

Долина освобождена.

**Нейтральная**

Игрок уходит, ничего не меняя.

**Властолюбивая**

Игрок забирает Сердце себе.

**Плохая**

Игрок погибает или становится частью проклятия.
