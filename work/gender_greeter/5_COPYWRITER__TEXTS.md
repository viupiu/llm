## Узел: Start

### Варианты ответа:
- "Как вас зовут?"

---

### UX ЛОГИКА:

**Memory:**
- Имя сохраняется из запроса пользователя (матчинг по словарю)
- Перезапрос имени при повторном входе не требуется

**Flow:**
- Линейный переход: Start → обработчик имени → конец
- Нет возможности пропустить шаги

**Reuse:**
- Вопрос "Как вас зовут?" уникален для этого бота
- Дублирование не требуется

**Links:**
- Не применимо

## Узел: Как вас зовут? - *

### Варианты ответа:
- "Ты просто супер!"
- "Супер!"
- "Ты просто супер! Рад встрече."

---

### UX ЛОГИКА:

**Memory:**
- Пол не определён (или неопознанный ввод)
- Имя не хранится

**Flow:**
- Fallback-узел, финальный, переход: конец
- Ловит унисекс, неопознанное, не имя

**Reuse:**
- Уникальный текст для fallback-ситуации

**Links:**
- Не применимо
