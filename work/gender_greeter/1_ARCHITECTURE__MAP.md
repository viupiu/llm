# Архитектурная карта: Gender Greeter

## Профиль целевой аудитории
Люди 18–45 лет, общаются в чате, вводят своё имя коротко, без лишнего текста.

---

## Навыки

### Skill: main
Основной навык бота.

#### Uz2: Start (root)
Тип узла: EVENT  
Задаёт вопрос: "Как вас зовут?"  
Цель: Старт диалога, запрос имени.  
Условия: Нет  
Переходы: ожидание ввода пользователя  
Якорь: [%that_anchor="Как вас зовут?"]  
Режим: DL  

#### Узел: Как вас зовут? - Мужское имя
Тип узла: DL  
Цель: Обработка мужского имени, ответ "ты красавчик".  
Условия: %that_anchor="Как вас зовут?"  
Скелет правила: `* [dict(male_names)] *`  
Референция: нет (не нужно вытаскивать имя)  
Переходы: конец  
Разграничение: только мужские имена, не женские, не унисекс  

#### Узел: Как вас зовут? - Женское имя
Тип узла: DL  
Цель: Обработка женского имени, ответ "ты красотка".  
Условия: %that_anchor="Как вас зовут?"  
Скелет правила: `* [dict(female_names)] *`  
Переходы: конец  
Разграничение: только женские имена, не мужские, не унисекс  

#### Узел: Как вас зовут? - *
Тип узла: Fallback  
Цель: Всё остальное (унисекс, неопознанное, не имя), ответ "ты просто супер".  
Условия: %that_anchor="Как вас зовут?"  
Правило: `*`  
Переходы: конец

## Логика сценария
1. Load → Start → "Как вас зовут?" → [%that_anchor="Как вас зовут?"]
2. Пользователь вводит имя:
   - Мужское → "Как вас зовут? - Мужское имя" → "ты красавчик" → конец.
   - Женское → "Как вы зовут? - Женское имя" → "ты красотка" → конец.
   - Иное → "Как вас зовут? - *" → "ты просто супер" → конец.
