# Бриф: Генератор оправданий

**BotSlug:** `excuse_gen_bot`
**Канал:** чат
**Режим оркестрации:** Interactive

## Бизнес-задача

Простой разговорный бот, генерирующий смешные, абсурдные или «правдоподобно-нелепые» оправдания для разных бытовых и рабочих ситуаций. Основная цель — вариативность и качество генерации. Проект призван протестировать конвейер фабрики чатботов на маленьком, но «живом» разговорном боте.

## Интенты (ситуации)

| Intent | Описание | Примеры запросов |
|--------|----------|-----------------|
| `excuse_late` | Опоздание | «почему я опоздал», «опоздал на встречу» |
| `excuse_task` | Не сделала задачу / не сдал отчет | «почему не сделал задачу», «почему не сдал отчет» |
| `excuse_absence` | Отсутствие | «почему не пришел», «почему не появился» |
| `excuse_silence` | Не ответил | «почему не ответил», «почему молчал» |
| `unknown` | Fallback | всё остальное |

## Требования к поведению

- Понимать тип ситуации (intent classification).
- Генерировать короткое смешное оправдание.
- Избегать повторов (anti-repeat mechanism).
- Менять стиль ответа (абсурдный, псевдо-правдоподобный, драматичный и т. д.).
- Поддерживать лёгкий разговорный тон.
- Без сложного сюжета и длинных диалогов.

## Архитектурные ожидания

- Минимальная структура: Start → Intent Classification → Ответ (генерация) → возможность повторного запроса.
- Anti-repeat: счётчик или история использованных оправданий в сессии.
- Fallback на непонятные запросы.

## Что не нужно

- Сложные ветки диалога.
- Логирование, CRM, интеграции.
- Голосовой канал.

## Нужен ли ML-автор

Да — для обучения IC по 4 интентам (`excuse_late`, `excuse_task`, `excuse_absence`, `excuse_silence`).
