# excuse_gen_bot — Тексты копирайтера

Все тексты — русские, неформальные, самодостаточные, готовы к закидыванию в udict для anti-repeat.

## Узел: Загрузка

### Варианты ответа:
- "Эй, привет! 🫵 Я — твоя личная фабрика отмазок. Работает без выходных, без совести и без судейства. Давай, рассказывай — что за беду надо прикрыть?"
- "Здарова! Ты попал к мастеру легенд. Генерирую оправдания для любой ситуации: отмажусь от босса, спасу от допроса, выкручусь из любой жопы. Чем могу помочь?"
- "О, ещё один человек в беде! 😏 Паниковать не надо — я тут именно для этого. Рассказывай, где завяз, придумаем что-то гениальное."
- "Привет, преступник. Шучу. Ну, почти. Я — генератор оправданий для всех, кому иногда надо выкрутиться. Так что ты намедни натворил?"

---

### UX ЛОГИКА:

**Memory:** не нужна — первая точка входа, старт конвеерса.
**Flow:** приветствие → сразу к `Какая у тебя беда?` без лишних кликов.
**Reuse:** текст приветствия можно ротировать для антитрепа, но логика одна.
**Links:** → `Какая у тебя беда?`

## Узел: Какая у тебя беда?

### Варианты ответа:

- "Окей, давай по порядку. Что случилось — опоздал, задачу не сделал, вовсе пропал или просто промолчал?"
- "Знаю, знаю, было такое у всех 😌 Ладно, выбери свою ситуацию из списка внизу — и держи отмажку."
- "Передохни. Сейчас будет лучше. Выбери свой тип бедствия 👇"

---

### UX ЛОГИКА:

**Memory:** якорь — пользователь уже в процессе выбора, нужно держать фокус на категории.
**Flow:** вопрос → ожидание интента → маршрутизация на соответствующий обработчик.
**Reuse:** базовый узел-хаб, тексты можно ротировать для разнообразия повторных обращений.
**Links:** → `Опоздание`, `Не сделал задачу`, `Отсутствие`, `Не ответил`, fallback `\*`

### UX ЛОГИКА:

**Memory:** udict хранит индекс показанной отмазки — next request отдаёт новый вариант без повторов.
**Flow:** пользователь интент `excuse_late` → случайная отмазка из списка → предложение «Хочешь ещё?».
**Reuse:** все варианты — независимые строки udict. После показа помечаются как «использованные», цикл продолжается до полного обхода.
**Links:** → кнопка «Ещё!» → повторная выдача из того же узла; кнопка «Другая беда» → `Какая у тебя беда?`
