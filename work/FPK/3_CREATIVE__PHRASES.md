# FPK — Фразы пользователей и скрипты бота

**BotSlug:** `FPK`
**Голос:** женский
**Назначение:** обучение DL-моделей классификации интентов (A/B/C), словари `keywords_*`, `user_type_*`

---

## 1. Интент A — Номер / статус обращения

Словарь: `keywords_appeal`.
Ловит: номер обращения, статус рассмотрения, «где мой запрос», трекинг заявки в ФПК.
Не должен ловить: билеты, расписание, отмена рейса.

1. Хочу узнать номер моего обращения
2. Подскажите номер моей заявки
3. Какой номер получил мой запрос?
4. Сможете сказать номер обращения?
5. Не знаю номер заявки, помогите найти
6. Номер заявки утерял, восстановите, пожалуйста
7. Скажите, пожалуйста, номер моего обращения
8. Хочу узнать статус моего обращения
9. какой статус по моей заявке
10. Какую дату рассмотрения дали по обращению?
11. Долго ли рассматривают жалобы?
12. Моё обращение уже в работе?
13. Где можно отследить ход по заявке?
14. Что происходит с моей жалобой?
15. Проверьте, пожалуйста, статус моего обращения

## 2. Интент B — Поезда и билеты

Словарь: `keywords_tickets`.
Ловит: отмена/перенос рейса, бронь, покупка/возврат билетов, расписание, движение поездов.
Не должен ловить: статус письма в delo@, номер обращения в ФПК.

1. Поезд отменили, что делать?
2. Отменили ли мой рейс?
3. Поезд задерживается сегодня?
4. Хочу купить билет на электричку
5. Как вернуть неиспользованный билет?
6. Оформите возврат билета, пожалуйста
7. Хочу забронировать место в поезде
8. Есть свободные места на завтра?
9. Какое расписание на Казань?
10. Подскажите, когда отправляется поезд?
11. Хочу перенести билет на другой день
12. Можно ли проехать по старому билету?
13. Отмена маршрута Москва — Нижний, правда?
14. Где узнать расписание на выходные?
15. Билет не подошёл, хочу обменять

7. Какие документы нужны для ребёнка?

## 7. Скрипты бота (финальные тексты, женский голос)

| Узел | Реплика бота |
|------|-------------|
| **1 — Приветствие** | Здравствуйте. Вы позвонили в Федеральную пассажирскую компанию. |
| **2 — Чем помочь?** | Чем я могу вам помочь? |
| **3 — Кто вы?** | Подскажите, пожалуйста, вы обращаетесь как физическое лицо или как юридическое лицо? |
| **4 — Справочная 8-800** | Поняла вас, по данному запросу вам необходимо обратиться по телефону справочной службы: 8 (800) 775-00-00, перевести вас? |
| **5 — Email delo@** | Поняла вас, по данному запросу вам необходимо отправить запрос на официальный электронный адрес компании: delo@fpc.ru, могу вам помочь чем-нибудь еще? |
| **6 — Прощание** | Рада была помочь. До свидания! |
| **7 — Исчерпание помощи** | К сожалению, ничем больше не могу вам помочь, большое спасибо за ваш звонок! Хорошего дня! |
| **G_AnythingElse** | Могу вам помочь чем-нибудь еще? |

## 8. Конфликтные примеры (явные границы для DL)

| Фраза | Интент | Почему |
|-------|--------|--------|
| «статус моего обращения» | **A** | обращение/заявка/жалоба в ФПК |
| «статус моей заявки» | **A** | то же |
| «статус билета» | **B** | вопрос про билет, не про обращение |
| «статус поеду» (ASR-ошибка «поезда») | **B** | контекст — поезд |
| «вернуть обращение» | **A** | действие над обращением |
| «вернуть билет» | **B** | действие над билетом |
| «где моя заявка» | **A** | трекинг заявки |
| «где мой поезд» | **B** | движение поезда |
| «мой запрос не рассматривают» | **A** | статус рассмотрения |
| «хочу обратиться с жалобой» | **C** | новая жалоба, а не трекинг существующего |
| «хочу пожаловаться на проводника» | **C** | не трек обращений, не билеты |
| «номер моего билета» | **B** | билётная тематика |
| «номер моего обращения» | **A** | обращение в ФПК |
| «перевести на операторов» | **C** | общий запрос, не A, не B |
