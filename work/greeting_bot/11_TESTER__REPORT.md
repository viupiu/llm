3. "поздравить, неважно для кого" → то же: отсутствует `{{хочу/нужно}}` для правила 3, отсутствует «поздравление» для правил 1 и 2

## РЕКОМЕНДАЦИИ

### Г2 — Для мужчин (1 непокрытая фраза)
**Проблема:** Слово «дядя» (побочная форма от «дед», но без «ь» → `дядьк~` его не покрывает).
**Решение (Автор Правил):** Добавить `дядя` в словарь `keywords_greeting_men`.

### Г13 — Универсальное (3 непокрытые фразы)
**Проблема:** Фразы «поздравить надо», «поздравить, просто поздравить», «поздравить, неважно для кого» — глагол «поздравить» без модификаторов «хочу/нужно» и без существительного «поздравление».
**Решение (Автор Правил):** Добавить правило `* поздравит ~ *` (без требования `{{хочу/нужно}}`), либо расширить синоним-сет `{{хочу/нужно}} → {{хочу/нужно/надо}}`.

### Г15 — Прощание (13 непокрытых фраз)
**Проблема:** Голый токен «до» не покрыт ни одним правилом. Формы «до, у меня время», «до, мне пора» и т.д. — все сводятся к отсутствию «до» в словаре.
**Решение (Автор Правил):** Добавить `до` в словарь `bye_solong` (как standalone token).
**Примечание:** фразы вида «до, у меня время», «до, мне нужно идти» — нестандартный/амбивалентный пользовательский ввод. Тестировщик фиксирует факт непокрытия. Решение «игнорировать» принимает пользователь при ревью отчёта.
