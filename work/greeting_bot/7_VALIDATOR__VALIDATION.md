### That Anchor Flow

```
G10 → %that_anchor="question_target"  →  G12/G13/G14/G15/G16 проверяют %that_anchor
   ↓
* fallback на G10 → повтор вопроса → цикл до валидного ответа
```

✅ Цепочка корректна. `%that_anchor` устанавливается в G10, проверяется в G12-G16, используется в fallback `*` на G10.
