# Архитектурная карта: Долина Туманных Руин

**Проект:** `misty_ruins_quest`
**Тип бота:** Текстовый квест (чат)
**Цель:** Пройти через Долину Туманных Руин, найти Древнее Сердце и решить судьбу долины.
**Статус:** approved

---

## Профиль целевой аудитории

Игроки текстовых квестов, 16-40 лет. Средний уровень знакомства с настольными RPG и текстовыми приключениями. Стиль общения на русском языке, без сленга, без командного синтаксиса. Ожидают атмосферные описания, последствия решений и несколько концовок.

---

## Флаги состояния

| Флаг | Устанавливается | Влияет на |
|------|----------------|-----------|
| `map_found` | Лес: загадка или Дорога: телеги | Подсказка в Крепости |
| `blessing` | Деревня: колодец | Спуск в Храм успешно |
| `guard_deceived` | Болото: Страж обманут | Дополнительный вариант на Мосту |
| `temple_records_read` | Пещеры: механизмы | Трансцендентная концовка в Зале Сердца |
| `artifact_obtained` | Пещеры: поиск артефакта (требует temple_records_read) | Упоминается в финале |

## Точки слияния

| Точка | Узел | Сходятся |

| Перекресток Воронов | `Crow_Question` | Лес + Дорога |
| Каменный Мост | `Bridge_Question` | Болото (shortcut + обход) |

## Сводная диаграмма переходов

```
EVENT Загрузка → "Войти в Долину?"
  ├─ [лес] → "Лес Шепчущих Корней?"
  │   ├─ [загадка ✓] → (%map_found) → "Перекрёсток Воронов?"
  │   └─ [мимо]      → "Перекрёсток Воронов?"
  │
  └─ [дорога] → "Старая дорога?"
      ├─ [телеги] → (%map_found) → "Перекрёсток Воронов?"
      └─ [мимо]   → "Перекрёсток Воронов?"

"Перекрёсток Воронов?" (слияние 1: Лес + Дорога)
  ├─ [уйти]    → End_Neutral
  └─ [вперёд]  → "Заброшенная Деревня?"

"Заброшенная Деревня?"
  ├─ [призрак]  → "Болото Светляков?"
  ├─ [колодец]  → (%blessing) → "Болото Светляков?"
  └─ [мимо]     → "Болото Светляков?"

"Болото Светляков?"
  ├─ [через болото] → "Болотный Страж?"
  │   ├─ [договориться] → "Каменный Мост?"
  │   ├─ [обмануть]  → (%guard_deceived) → "Каменный Мост?"
  │   └─ [силой]     → End_Bad
  └─ [обойти]     → "Каменный Мост?"

"Каменный Мост?" (слияние 2: shortcut + обход)
  ├─ [договориться]              → "Горный Проход?"
  ├─ [обмануть с guard_deceived] → "Горный Проход?"
  ├─ [обмануть без флага]        → "Болото Светляков?" (цикл с выходом)
  └─ [силой]                     → End_Bad

"Горный Проход?"
  ├─ [пещеры]   → "Хрустальные Пещеры?"
  │   ├─ [механизмы] → (%temple_records_read) → "Артефакт?"
  │   │   ├─ [найти] → (%artifact_obtained) → "Крепость Без Короля?"
  │   │   └─ [мимо]  → "Крепость Без Короля?"
  │   └─ [мимо]      → "Крепость Без Короля?"
  └─ [перевал]  → "Крепость Без Короля?"

"Крепость Без Короля?" (слияние 3: Пещеры + Перевал)
  ├─ [с blessing]   → "Зал Сердца?"
  └─ [без blessing] → End_Bad

"Зал Сердца?"
  ├─ [уничтожить]     → End_Good
  ├─ [забрать]        → End_Power
  ├─ [оставить]       → End_Neutral
  └─ [осознать, с temple_records_read] → End_Transcendence
```
